# 🎉 Event Decors

A modern and responsive event decoration website built using HTML and CSS.  
This project showcases elegant event designs, decoration themes, and a clean user interface for event planning businesses.

---

## 📌 Overview

Event Decors is a static frontend website designed for showcasing event decoration services. The project focuses on creating an attractive and responsive user experience using only HTML and CSS.

The website can be used for:
- Wedding decoration showcases
- Birthday event themes
- Corporate event decoration
- Portfolio display for decorators

---

## ✨ Features

- 🎨 Modern UI Design
- 📱 Fully Responsive Layout
- 🖼 Image Gallery Section
- 🧾 Service Showcase
- 📞 Contact Section
- 🧭 Smooth Navigation
- 💻 Clean HTML & CSS Structure
